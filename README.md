# Zeta visualizer
![](https://cabbache.github.io/zetaviz.gif)

Has a cache for panning more efficiently.  
Make sure to use a high enough precision (> 800) to see the zeroes correctly.

## Controls
```
Controls:
  Mouse Drag: Pan view
  Mouse Wheel: Zoom in/out
  +/- : Increase/decrease precision
  [/] : Decrease/increase computation resolution
  A : Toggle axes
  L : Toggle logarithmic color scale
  F : Toggle filter (hide Re >= 1)
  R : Reset view
  Space : Force re-render
  ESC : Exit
```
